# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] — 2026-07-21

### Added

- Project skeleton with FastAPI backend, React frontend, and Docker Compose
- JWT authentication (register, login)
- CSV file upload with metadata extraction
- Dataset listing per user
- PostgreSQL database with Alembic migrations
- Dark/light mode UI with Shadcn components
- Swagger API documentation
- Pytest test suite for auth and dataset endpoints
